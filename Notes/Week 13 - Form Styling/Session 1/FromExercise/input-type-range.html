Using the Range Input type

The range input type indicates to the browser that the field is expecting a numeric value selected from within a given range.

This input type can also be used with the min, max, and step attributes (number intervals) to provide additional UI features to help the user select a number. The slider option looks a bit odd in that no values are displayed, but may be useful for more 'analog' inputs. Step defaults to 1 and can be left out in most cases.

Notice the position of the sliders with the given values they already contain. Change their values and see what happens.
