Using the autocomplete Attribute

Some browsers provide native support for remembering the values of form fields and displaying those values to the user to speed data entry later.

  • Autocomplete Settings
  • autocomplete="on"
  • autocomplete="off"

The autocomplete attribute allows you to control whether a form field remembers its value. By default, autocomplete is set to "on". Fill out the form, submit it, and try filling it out again to see what happens. Now change the form so that the input values will not autocomplete for the user.
