Using the New Input Types

Create form fields (labels and inputs) for each of the input types listed below. Don't forget to include the 'for', 'name', and 'id' attributes. What happens when you fill out and submit the form?

  • Type = search
  • input type="search"
  • Type = email
  • input type="email"
  • Type = color
  • input type="color"
  • Type = url
  • input type="url"
  • Type = telephone
  • input type="tel"
  • Type = password
  • input type="password"
  • Type = number
  • input type="number" min="1" max="100"
  • Type = time
  • type="time"
  • Type = date
  • type="date"
  • Type = week
  • type="week"
  • Type = month
  • type="month"
