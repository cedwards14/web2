HTML5 Forms Activity

An HTML form is a main of a document containing normal content, markup, special elements called controls (checkboxes, radio buttons, menus, etc.), and labels on those controls. Users generally "complete" a form by modifying its controls (entering text selecting menu items, etc.), before submitting the form to an agent for processing (e.g., to a Web server, to a mail server, etc.).
